# NoteTalkingApp
Create an application to save Notes for Users   

Use Case:
1- A system needs to store the notes of different users. 
2- After a user successfully logins into the application, the API should
return recent 10 notes. 
3- The system needs to delete all the notes other than last 10 recent notes
on the hourly basis.  

Validations to be considered:
1- The notes will only allow [@, ; & * + -] special characters. 2- The Notes cannot be more than 500 letters.
3- Allow User to delete the notes  

Technologies to be considered: Java 8 , Spring Boot , JPA , Angular.
